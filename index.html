    Seam Carving

    Resizing image based on content-awareness (Python & C)

    Seam-carving is a content-aware image resizing technique where the image is reduced in size by one pixel of width (or height) at a time. A vertical seam in an image is a path of pixels connected from the top to the bottom with one pixel in each row; a horizontal seam is a path of pixels connected from the left to the right with one pixel in each column. Unlike cropping and scaling a photo, seam carving preserves important features such as aspect ratio and set of objects present. Essentially it takes out the "unimportant" parts of a photo. The gif on the left continues to remove the less important seams, leaving the humans till the end.

    Autocomplete

    Completes a word given a string of initial letters ( C )

    Completes a given substring (partial word) based on relevance in the English language. For example, if the inputted substring is "th", the outputed string will be "the", as it is the most used word in english starting with "th". The worst case time complexity at most is log(10 000), as there are 10 000 words stored in the file stored in lexogrpahic order, and the searching algorithm is binary search.
